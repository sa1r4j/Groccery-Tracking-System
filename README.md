# Groccery-Tracking-System
This Grocery Store Inventory System is a simple web-based application developed using HTML, CSS, and JavaScript to manage grocery stock. It allows users to add, view, and delete inventory items while storing data. The system also highlights low-stock items and performs basic inventory management efficiently.
## How It Works

This GroceryTrack system is a web-based grocery inventory management application that allows users to register and log in, then manage grocery stock through a secure inventory dashboard. Users can add, edit, delete, and search inventory items, while the system automatically tracks total inventory value, shows low-stock alerts, and displays stock statistics on the dashboard. The frontend is built with HTML, CSS, and JavaScript, while a Node.js + Express server handles API requests and stores data in a JSON file.   

## How to Use

1. Clone or download the project files.
2. Install dependencies using `npm install`.
3. Start the server using `npm start`.
4. Open `http://localhost:3000` in your browser. 

### Steps to Use the Application

* Register a new account or log in with existing credentials. 
* Go to the **Inventory** page to add grocery items with name, quantity, and price. 
* Use **Edit** or **Delete** to manage stock. 
* Use the **Search** bar to find items quickly. 
* Open the **Dashboard** to view inventory statistics, stock levels, and low-stock alerts. 

This system is designed to help users monitor and manage grocery inventory in a simple and efficient way.
